[1;37mSYNOPSIS:[0m
    Execute a query on multiple databases to generate a tsv format report.
    The query can be parallelized on multiple databases.

[1;37mUSAGE:[0m dbQueryAllDatabases [OPTIONS] [ARGUMENTS]
[1;37mUSAGE:[0m dbQueryAllDatabases [--jobs|-j <jobsCount>] [--bar|-b]
  [--help|-h] [--config] [--bash-framework-config <bash-framework-config>]
  [--verbose|-v] [-vv] [-vvv] [--env-file <env-file>] [--log-level <log-level>]
  [--log-file <log-file>] [--display-level <display-level>] [--no-color]
  [--theme <theme>] [--version] [--quiet|-q] [--separator|-s <separator>]
  [--from-dsn|-f <dsn>]

[1;37mARGUMENTS:[0m
  [1;34margQuery[0m {single} (mandatory)
    Query to execute 
      - <file>, try to execute the mysql query provided by the file

      - <queryFile>, search for query file in queries directory (see below)

      - else the argument is interpreted as query string

  [1;34m--jobs[0m, [1;34m-j <jobsCount>[0m {single}
